# Engineering Tool

The tool is a physic formula for Engineering or Science : **Python 3.6+**.


# Features
- Area Formula
- Volume Formula
- Electric Formula
- Pressure Formula
- Temperature Formula  


# Example
    from engineering_tool.electrics import Power
    voltage = 15 # Volt
    current = 2.5 # Amp
    power = Power.VoltageAndCurrent(voltage,current)
    print("Power is %.2f Watt"%power)
